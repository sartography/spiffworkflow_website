SpiffWorkflow

SpiffWorkflow is an open source Python library with source code available on GitHub and detailed documentation available on ReadTheDocs. It is the core library within SpiffArena. Its target audience is Python developers who want to create custom workflow systems or embed BPMN features into existing applications.
